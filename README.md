## Online Sınav Sistemi

Bu proje, web tabanlı bir online sınav sistemi geliştirmek amacıyla yapılmıştır.
Öğrencilerin sınav sorularını cevaplayabildiği ve sınav sonucunu görüntüleyebildiği basit bir sistemdir.

Proje, backend – frontend – veritabanı ilişkisini öğrenmek ve uygulamak için geliştirilmiştir.

## Özellikler

Sınav sorularını veritabanından çekme

Çoktan seçmeli sorular (A, B, C, D)

Öğrenci cevaplarının alınması

Doğru / yanlış sayısının hesaplanması

Sınav sonucunun ekranda gösterilmesi
## Çalışma Mantığı

Sınav soruları SQL Server veritabanından alınır

Sorular exam.html sayfasında öğrenciye gösterilir

Öğrencinin verdiği cevaplar backend’e gönderilir

Doğru cevaplarla karşılaştırma yapılır

Sonuçlar result.html sayfasında görüntülenir
## Geliştirilebilecek Özellikler

Kullanıcı girişi (login sistemi)

Birden fazla sınav desteği

Zaman sınırlı sınav

Detaylı sonuç ekranı

Öğretmen paneli

## Kullanılan Teknolojiler

Node.js (Express.js) – Backend

SQL Server – Veritabanı

HTML / CSS / JavaScript – Frontend
